# weatherdashboardpoc
 Weather Dashboard for getting the wether of a entered city, uses the openweathermap API - POC

 # Setup
 npm start

 # APIs used
 URL: https://api.openweathermap.org/data/2.5/weather?q={CITY_NAME}&appid={API_KEY}
